# Mad Muscles Web App

Responsive marketing and funnel web app for Mad Muscles, built with React, TypeScript, Vite, and Tailwind CSS.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- tsParticles

## Getting Started

```sh
npm install
npm run dev
```

Open `http://localhost:8080`.

## Build

```sh
npm run build
npm run preview
```
